Theorem 51 Dusart
#

There exists a constant \(X_0\) (one may take \(X_0 = 89693\)) with the following property: for every real \(x \ge X_0\), there exists a prime \(p\) with

\[ x {\lt} p \le x\Bigl(1 + \frac{1}{\log ^3 x}\Bigr). \]
